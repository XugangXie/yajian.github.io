WeakHashMap

WeakHasMap是针对弱引用的一个类，深入理解java虚拟机中对java引用概念进行了详细说明，在此不做重复解释。需要注意的是弱引用只存活到下一次垃圾回收之前，WeakHasMap中的键都是弱引用类型，所以该类可以自动移除已经被回收的键所对应的k-v对。

  • 该类支持null值输入
  • 默认每次扩容一倍
  • fail-fast

类定义

继承了AbstractMap类，实现了Map接口，注意该类没有实现cloneable、serializable接口
